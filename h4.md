# Harjoitus 4

## Harjoituksen tarkoitus

Harjoituksen tarkoituksena on vuokrata julkinen palvelin ja sille nimi.

Sainkin jo tunnilla tehtyä vuokrauksen ja domain varauksen. Itse palvelimen vuokrasin Digitaloceanilta ja nimen NameCheapilta. Palvelimen speksit alla olevassa kuvassa. Domainin osoite taas on leikas.me 

![Image](/palvelin/palvelin.png "default")

Pääsin palvelimeen omalta virtuaalikoneeltani SSH:n kautta. Ajoin sille tarvittavat komennot; palomuurin asennus, reiät palomuuriin, Apache2 asennus, palomuurin käynnistys.

>sudo apt-get install ufw
>
>sudo ufw allow 22/tcp
>
>sudo ufw allow 80/tcp
>
>sudo ufw enable
>
>sudo apt-get install apache2
